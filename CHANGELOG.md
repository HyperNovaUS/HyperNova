# Changelog

All notable changes to HyperNova will be documented in this file.

## [0.4.2] - 2026-07-15
### Added
- Virtuals agent execution engine and bridge adapter
- Enhanced solver anti-cheat with canary probes
- Rate limiting middleware for API server
- Testnet configuration for Robinhood Chain Sepolia

### Fixed
- Intent deadline validation edge cases
- Bridge adapter gas estimation for large transfers
- Dashboard chart rendering on mobile

## [0.4.1] - 2026-07-12
### Added
- Cross-chain transaction monitoring dashboard
- Batch intent submission support
- New solver: Virtuals AI Agent integration

### Changed
- Upgraded Solidity to 0.8.20
- Optimized gas costs by 15%

## [0.4.0] - 2026-07-08
### Added
- Virtuals Protocol integration
- EVMBridgeAdapter for CCIP compatibility
- Robinhood Chain Staking contract (`RHStaking.sol`)
- Token contract (`HyperNovaToken.sol`)
